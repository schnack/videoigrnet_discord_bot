Бот оповещает о изменениях на сайте videoigr.net.


Добавление категории:

    [vgnet add <url>
			
Просмотр списка категорий:
    
    [vgnet list
			
Удаление категории:
    
    [vgnet del <num>

Запуск уведомлений:
    
    [vgnet start
			
Остановка уведомлений
    
    [vgnet stop

Посмотреть статус уведомлений в текущем канале
    
    [vgnet status
